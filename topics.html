// LASTING HALL - Topics スプレッドシート公開スクリプト v3 // G列に「TRUE」またはチェックボックス（✓）が入った行だけ公開 // 画像URLはGoogleドライブの共有URLを自動変換 function doGet(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); const data = sheet.getDataRange().getValues(); const topics = data.slice(1) .filter(row => { const flag = row[6]; const isPublished = flag === true || String(flag).trim().toUpperCase() === 'TRUE'; const hasTitle = row[2] && String(row[2]).trim() !== ''; return isPublished && hasTitle; }) .map(row => ({ date: row[0] ? Utilities.formatDate(new Date(row[0]), 'Asia/Tokyo', 'yyyy.MM.dd') : '', category: String(row[1] || '').trim(), title: String(row[2] || '').trim(), desc: String(row[3] || '').trim(), image: convertDriveUrl(String(row[4] || '').trim()), url: String(row[5] || '').trim() })); const output = ContentService.createTextOutput(JSON.stringify(topics)); output.setMimeType(ContentService.MimeType.JSON); return output; } // GoogleドライブのURLを画像表示用URLに自動変換 function convertDriveUrl(url) { if (!url) return ''; // 既に変換済みの場合はそのまま返す if (url.includes('uc?export=view')) return url; // https://drive.google.com/file/d/【ID】/view 形式 const match1 = url.match(/\/file\/d\/([a-zA-Z0-9_-]+)/); if (match1) { return 'https://drive.google.com/uc?export=view&id=' + match1[1]; } // https://drive.google.com/open?id=【ID】 形式 const match2 = url.match(/[?&]id=([a-zA-Z0-9_-]+)/); if (match2) { return 'https://drive.google.com/uc?export=view&id=' + match2[1]; } // その他のURL（外部画像など）はそのまま返す return url; }
